# 議事録君 - 配布マニュアル

## 使い方（社内配布用）

このフォルダには「議事録君」のすべてのファイルが含まれています。
以下のいずれかの方法でご利用ください。

### 方法1: 社内Webサーバーで公開する（推奨）
このフォルダの中身を、社内のWebサーバー（IIS, Apache, Nginxなど）の公開ディレクトリに配置してください。
URLを共有するだけで、ブラウザからアクセスできます。

- 推奨ブラウザ: Google Chrome（音声認識・画面共有機能のため）
- 注意: `file://` プロトコル（ファイルを直接ダブルクリックして開く）では、ブラウザのセキュリティ制限によりマイクやカメラが動作しない場合があります。必ず `http://` または `https://` でアクセスしてください。

### 方法2: 簡易ローカルサーバーで実行する
管理者やエンジニアの方がローカルで試す場合は、以下の手順で起動できます。

1. Pythonがインストールされている場合（コマンドプロンプトで実行）:
   ```cmd
   python -m http.server 8000
   ```
   ブラウザで `http://localhost:8000` を開きます。

2. Node.js (http-server) を使う場合:
   ```cmd
   npx http-server .
   ```

## 機能概要
- **面接モード**: 履歴書分析・評価シート・深掘り質問機能
- **MTGモード**: 議事録作成・マイク入力・画面共有音声入力
- **Topics分析**: 会話から重要なトピックと質問をAI抽出
- **データ保存**: すべてブラウザ内に保存（サーバー送信なし）

## 利用上の注意
- APIキー設定: 初回起動時に右上の「設定」からGemini APIキー（必須）、Groq APIキー（任意）を入力してください。
- データは各ユーザーのブラウザごとのlocalStorageに保存されます。
